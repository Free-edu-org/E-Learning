# Project Overview: FreeEdu e-learning

## Opis
Platforma do nauki języka angielskiego z naciskiem na nowoczesne metody interaktywne i rozpoznawanie mowy.

## Stack Technologiczny
- **Frontend:** React 19 + TypeScript + Vite.
- **Backend:** Java 25 + Spring Boot 4.0.1.
- **Komunikacja:** REST API (WebFlux).
- **Baza danych:** MySQL + Flyway (migracje).
- **Inne:** Spring Security (JWT), MapStruct.

## Kluczowe Moduły
1. **Auth:** Obsługa JWT i ról użytkowników.
2. **Task Generator:** Silnik zadań (Matching, Gap-fill, Multiple choice).
3. **STT Integration:** Integracja z modelem Whisper (Python) do zadań ustnych.
