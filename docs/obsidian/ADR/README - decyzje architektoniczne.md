# ADR

ADR przechowuje decyzje, ktore warto pamietac po kilku tygodniach pracy. Krotki indeks jest w [[Decision log]].

## Zasada

- Jeden plik = jedna decyzja.
- Numeruj rosnaco: `0001`, `0002`, `0003`.
- Statusy: `proposed`, `accepted`, `deprecated`, `superseded`.
- Opisuj konsekwencje, nie tylko sama decyzje.
